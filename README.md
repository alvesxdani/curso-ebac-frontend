# Formulário Simples com Máscaras (HTML, CSS, JavaScript e jQuery)

Este projeto é um formulário simples que utiliza HTML, CSS, JavaScript e jQuery. Além disso, inclui plugins jQuery para máscara de telefone e CPF, proporcionando uma experiência de preenchimento mais fácil e intuitiva.

## Conteúdo do Projeto

O projeto é composto pelos seguintes arquivos e diretórios:

- **index.html:** O arquivo HTML principal que contém a estrutura do formulário.
- **style.css:** O arquivo CSS para estilizar o formulário.
- **scripts.js:** O arquivo JavaScript que lida com a lógica do formulário.
- **jquery.min.js:** O arquivo jQuery, uma biblioteca JavaScript, que simplifica a manipulação do DOM.
- **jquery.maskedinput.min.js:** O plugin jQuery para adicionar máscaras aos campos de telefone e CPF.
- **images/:** Qualquer imagem ou mídia usada no formulário deve ser armazenada neste diretório.

## Como Utilizar o Projeto

1. Clone o repositório para sua máquina local:

   ```bash
   git clone https://github.com/alvesxdani/exercicios_ebac_frontend.git
   git checkout exercicio_plugins_jquery
   ```

2. Abra o arquivo `index.html` em seu navegador da web preferido.

3. Preencha o formulário, observe as máscaras automáticas para os campos de telefone e CPF.

## Funcionalidades

- **Máscara de Telefone:** O plugin jQuery adiciona automaticamente a máscara ao campo de telefone.
- **Máscara de CPF:** O plugin jQuery adiciona automaticamente a máscara ao campo de CPF.
- **Validação Simples:** Pode ser adicionada uma lógica de validação básica no arquivo `script.js`.
